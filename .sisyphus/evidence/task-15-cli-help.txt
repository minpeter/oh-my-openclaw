Usage: oh-my-openclaw [options] [command]

OpenClaw configuration preset manager

Options:
  -V, --version             output the version number
  -h, --help                display help for command

Commands:
  list [options]            List available presets
  apply [options] <preset>  Apply a preset to your OpenClaw configuration
  export [options] <name>   Export current OpenClaw configuration as a preset
  diff [options] <preset>   Show diff between current config and a preset
  help [command]            display help for command
Usage: oh-my-openclaw apply [options] <preset>

Apply a preset to your OpenClaw configuration

Arguments:
  preset       Preset name to apply

Options:
  --dry-run    Show what would change without applying
  --no-backup  Skip creating a backup (use with caution)
  -h, --help   display help for command
